Simulation Help Pop-up

Simulation items can carry their own overrides without changing your normal brush controls.

Override precedence

  • Selected simulation item overrides win whenever that override is set.
  • If an item has no override, the usual brush sidebar setting is used.
  • Changing brush controls never rewrites existing simulation-item overrides.

Boid guides

  • Every boid path contributes an attractive field at the same time.
  • Boids steer by vector-summing all enabled path and point attractions.
  • Per-path strength and radius let you bias one guide more than another.

Repulsion hardness

  • Repel points now include hardness, which changes how sharply repulsion ramps up near the source.
  • Lower hardness feels softer and wider; higher hardness feels tighter and stronger near the center.

Inspector workflow

  • Use the floating Simulation HUD to place spawn, attract, repel, path, edge, or pheromone items.
  • Select an item on the canvas or from the inspector list to edit it in the right-side overlay.
  • Disable an item to keep it in the scene without deleting it.

Brush behavior

  • Outside Simulation mode, boid and ant brushes keep using the normal sidebar settings.
  • Simulation mode layers per-item overrides on top of those brush defaults.
